# 🔐 JIT IAM System (Just-In-Time Identity Access Management)

A secure and efficient system to manage temporary access to resources using token-based authentication.

---

## 🚀 Features

- 🔑 Token-based secure access
- ⏳ Time-limited access (auto expiry)
- 🖥️ Multiple resource selection (VMs, Containers)
- ✅ Access approval workflow
- 📊 System logs tracking
- 🎨 Modern UI with animations

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- UUID for token generation

---

## ⚙️ How It Works

1. User requests access to a resource
2. System generates a unique token
3. Token is valid for limited time (5 mins)
4. User enters token to access resource
5. System validates and grants access
